三行布局说明

  • 第一行：显示总数和每页条数选择器
  • 第二行：分页导航按钮（上一页、页码、下一页）
  • 第三行：跳转输入框，可直接输入页码跳转

响应式测试

请调整浏览器窗口宽度到 480px 以下，查看三行布局效果。在宽屏下，分页组件会恢复为单行水平布局。

CSS 实现要点

/* 窄屏幕下的三行布局 */
@media (max-width: 480px) {
  .pagination-container .el-pagination {
    flex-direction: column;
    align-items: center;
    gap: 15px;
  }
  
  /* 第一行：总数和每页条数选择器 */
  .el-pagination__total,
  .el-pagination__sizes {
    text-align: center;
  }
  
  /* 第二行：分页导航按钮 */
  .el-pagination__prev,
  .el-pagination__next,
  .el-pager {
    display: flex;
    justify-content: center;
    align-items: center;
  }
  
  /* 第三行：跳转输入框 */
  .el-pagination__jump {
    text-align: center;
  }
}
